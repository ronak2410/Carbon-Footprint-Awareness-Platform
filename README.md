# AURA Carbon — Carbon Footprint Analytics Platform

Monitor, understand, and reduce your environmental footprint with personalized insights, historic tracking, and actionable habit adjustments.

## Features

- Real-time carbon footprint calculator (electricity, transport, LPG, diet)
- Sector breakdown charts and comparative analysis
- Habit simulation for reduction strategies
- Historical log tracking with CSV export
- Eco-awareness quiz with achievement badges
